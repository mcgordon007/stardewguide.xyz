# Stardew Valley Blue Grass 指南

## 简介

**Blue Grass（蓝草）** 是《星露谷物语》1.6 版本新增的一种特殊草类。与普通草相比，蓝草为农场动物提供更好的营养，可以增加动物产品的品质。

## 获取方式

- **购买**：从 Pierre General Store 或 JojaMart 购买 Blue Grass Starter（蓝草 starter）
- **价格**：比普通草 starter 稍贵
- **解锁条件**：1.6 版本后自动解锁

## 种植方法

1. 购买 Blue Grass Starter
2. 在农场空地上使用 starter 种植
3. 蓝草会像普通草一样蔓延生长
4. 需要留出空间让草自然扩散

## 效果与优势

- **提高动物产品品质**：食用蓝草的动物有更高概率产出高品质产品
- **增加动物心情**：食用蓝草的动物心情更好
- **替代干草**：可以完全替代干草作为动物饲料
- **冬季枯萎**：与普通草一样，蓝草在冬季会枯萎

## 与普通草的区别

| 特性 | 普通草 | 蓝草 |
|------|--------|------|
| 动物产品品质 | 标准 | 提升 |
| 动物心情 | 标准 | 更好 |
| 蔓延速度 | 正常 | 正常 |
| 冬季枯萎 | 是 | 是 |
| 价格 | 较便宜 | 稍贵 |

## 注意事项

- 蓝草在冬季会枯萎，需要提前准备干草或搭建暖房（Greenhouse）
- 动物会优先食用蓝草而不是干草（如果两者都可获得）
- 蓝草和普通草可以共存，但蓝草效果更好
- 需要定期清理过度蔓延的草以保持农场整洁

## 策略建议

- 在动物放牧区域专门种植蓝草
- 配合围栏（Fence）使用，控制蓝草的生长范围
- 在秋季末大量储存干草，以备冬季使用
- 对于以动物产品为主要收入的农场，蓝草是必投资的项目

## 相关物品

- **Grass Starter（草 starter）**：普通草的种子
- **Hay（干草）**：冬季或室内喂养的替代饲料
- **Silo（筒仓）**：储存干草的建筑

## 版本历史

- **1.6 版本**：新增草类型
