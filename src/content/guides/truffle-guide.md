# 松露攻略

松露（Truffle）是星露谷物语中最赚钱的农牧产品之一。通过养猪获取松露，再加工成松露油，可以实现极高的利润。

## 基本信息

| 属性 | 数据 |
|------|------|
| 售价 | 625g（基础品质） |
| 季节 | 春季和秋季（猪只在春/秋季觅食） |
| 获取方式 | 猪在户外觅食时产出 |
| 加工产品 | 松露油（Oil Maker，售价1,065g） |

## 如何获取松露

### 前提条件

1. 建造**豪华谷仓（Deluxe Barn）**（花费10,000g + 550木头 + 300石头）
2. 从玛妮处购买**猪（Pig）**，每头16,000g
3. 猪在春季和秋季的户外会自动觅食产出松露

### 觅食机制

- 猪每天会走出谷仓觅食
- 觅食等级影响松露产出范围
- 每头猪每天最多产出1个松露
- 冬季猪不会觅食（不产出松露）

## 觅食等级与产出范围

| 觅食等级 | 觅食范围 |
|----------|----------|
| 0级 | 无松露 |
| 1级 | 谷仓周围小范围 |
| 4级 | 中等范围 |
| 8级 | 大范围 |
| 最高级 | 最大范围（约11x11格） |

> 觅食等级越高，猪找到松露的概率和范围越大。

## 养猪策略

### 最佳配置

| 项目 | 建议 |
|------|------|
| 谷仓数量 | 1-3个豪华谷仓 |
| 每个谷仓猪数 | 12头（满员） |
| 谷仓位置 | 放在农场开阔区域 |
| 自动采集 | 配合自动采集器 |

### 收益估算

| 配置 | 日均收入（春/秋季） |
|------|---------------------|
| 4头猪 | 约2,500g |
| 8头猪 | 约5,000g |
| 12头猪 | 约7,500g |
| 24头猪（2谷仓） | 约15,000g |
| 36头猪（3谷仓） | 约22,500g |

> 以上为松露油+工匠职业的估算收入。

### 自动化建议

- 在谷仓门口放置**自动采集器**，自动收集松露
- 在农场放置**油机**，将松露加工成松露油
- 配合**出货箱**，自动出货

## 实用技巧

### 提升觅食等级
- 大量采集野外物品提升觅食等级
- 使用觅食技能书（技能书#3）
- 吃觅食相关食物

### 冬季管理
- 冬季猪不觅食，将猪关在谷仓内
- 确保谷仓内有足够的干草
- 可以在冬季提升觅食等级

### 谷仓布局
- 谷仓门朝向开阔区域
- 清除谷仓周围的障碍物
- 让猪有最大的觅食空间

## 常见问题

**Q: 猪在什么季节产出松露？**
A: 只有春季和秋季。冬季猪不会觅食，不产出松露。

**Q: 松露油比直接卖松露更赚钱吗？**
A: 是的，松露油售价1,065g，比松露基础价625g高70%。

**Q: 需要多少头猪？**
A: 建议至少4头猪开始，资金充足后扩展到12-36头。

**Q: 觅食等级怎么提升？**
A: 通过采集野外物品（野山葵、蘑菇、黑莓等）提升。

**Q: 松露可以在温室种吗？**
A: 不能，松露只能通过猪觅食获得，不能种植。
