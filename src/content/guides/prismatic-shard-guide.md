# 五彩碎片攻略

五彩碎片（Prismatic Shard）是星露谷物语中最稀有、最珍贵的物品之一。它有着多种重要用途，是游戏后期发展的关键道具。

## 获取方式

五彩碎片的获取概率极低，以下是所有获取途径：

### 采矿获取

| 来源 | 概率 | 说明 |
|------|------|------|
| 铱矿节点 | 3.5% | 骷髅洞穴150层以上 |
| 神秘矿石 | 25% | 矿井任意层数随机出现 |
| 采石场铱矿 | 4% | 采石场中的铱矿节点 |
| 宝石矿 | 极低 | 矿井中的宝石矿节点 |

### 怪物掉落

| 怪物 | 掉落概率 | 出现地点 |
|------|----------|----------|
| 暗影狂徒 | 0.05% | 矿井80-120层 |
| 暗影萨满 | 0.05% | 矿井80-120层 |
| 木乃伊 | 0.1% | 骷髅洞穴 |
| 紫色史莱姆 | 0.05% | 骷髅洞穴 |
| 铱蝙蝠 | 0.05% | 骷髅洞穴 |

### 其他获取方式

| 方式 | 概率 | 说明 |
|------|------|------|
| 钓鱼宝箱 | 极低 | 钓鱼时出现的宝箱 |
| Omni Geode | 0.4% | 万象晶球 |
|  淘盘 | 极低 | 河流中使用淘盘 |
|  骷髅洞穴宝藏房 | 中等 | 骷髅洞穴中的宝藏房间 |
|  沙漠矿车 | 极低 | 卡利科沙漠矿车奖励 |

> 获取五彩碎片最可靠的方式是在骷髅洞穴150层以上大量开采铱矿节点。

## 主要用途

### 1. 获取银河剑（最重要）

将五彩碎片带到卡利科沙漠的三根柱子中央：
- 获得 **银河剑**（游戏最强武器之一）
- 同时解锁银河匕首和银河锤的购买权限
- 在探险家公会购买

> 这是五彩碎片最重要的用途，强烈建议第一个五彩碎片用于获取银河剑。

### 2. 武器/工具附魔

在火山地牢第10层的锻造台：
- 消耗 **1个五彩碎片 + 20个火山晶石**
- 为武器或工具添加附魔效果
- 每个工具只能有一个附魔，但可以用新的五彩碎片替换

**武器附魔效果：**
| 附魔 | 效果 |
|------|------|
| 巧妙 | 攻击速度提升 |
| 虫子杀手 | 对虫类怪物伤害翻倍 |
| 十字军 | 对暗影怪物伤害翻倍 |
| 吸血鬼 | 击杀怪物时恢复生命值 |
| 干草 | 击杀史莱姆时掉落干草 |

**工具附魔效果：**
| 工具 | 推荐附魔 | 效果 |
|------|----------|------|
| 锄头 | 范围 | 增加耕作范围 |
| 水壶 | 省力 | 减少蓄力时间 |
| 斧头 | 丰富 | 增加木材产量 |
| 镐 | 迅捷 | 增加采矿速度 |
| 鱼竿 | 自动上钩 | 自动钓鱼 |
| 喷壶 | 无底洞 | 无限水量 |

### 3. 捐赠博物馆

- 可以捐赠给博物馆（Gunther）
- 但不是必须捐赠的物品
- 建议优先用于其他用途

### 4. 礼物

- 五彩碎片是**几乎所有村民的"最爱"礼物**（例外：Haley讨厌五彩碎片）
- 送给任何NPC都能获得最高好感度
- 但鉴于其稀有性，不建议作为普通礼物使用

### 5. 制作结婚戒指

- 在旅行商人处购买配方
- 需要 1个五彩碎片 + 5个铱锭
- 用于多人游戏中向其他玩家求婚

### 6. 裁缝

- 在缝纫机中使用五彩碎片可以制作特殊服装
- 可以染出独特的彩虹色

### 7. 任务

- 齐先生的"四颗宝石"特别任务需要收集4个五彩碎片
- 任务期间新收集的才算数

## 最佳使用策略

### 第一个五彩碎片
**强烈推荐用于获取银河剑**。银河剑是游戏最强武器，大幅提升战斗能力。

### 第二个及以后的五彩碎片
1. **优先附魔工具**：推荐先附魔镐（迅捷）和水壶（省力）
2. **附魔武器**：推荐"巧妙"（攻速）或"吸血鬼"（回血）
3. **备用**：留1-2个备用，用于任务或紧急情况

### 不建议的用途
- 捐赠博物馆（不是必须的）
- 作为普通礼物（太浪费了）
- 出售（售价2,000g，但价值远高于售价）

## 刷五彩碎片技巧

### 最佳地点：骷髅洞穴
1. 大量制作楼梯，快速下到150层以上
2. 使用炸弹炸开铱矿节点
3. 击杀紫色史莱姆和木乃伊
4. 寻找宝藏房间

### 幸运加成
- 选择高幸运日进入骷髅洞穴
- 吃香辣鳗鱼（+1运气）
- 佩戴幸运戒指
- 携带幸运兔脚

### 效率提升
- 使用无限之刃或银河剑快速击杀怪物
- 带足炸弹和楼梯
- 忽略普通矿石，专注找铱矿节点

## 1.6更新变化

- 新增更多附魔选项
- 龙牙附魔可以与武器附魔叠加
- 齐先生的"四颗宝石"任务需要五彩碎片

## 常见问题

**Q: 五彩碎片能卖多少钱？**
A: 售价2,000g，但强烈不建议出售，用途远比售价有价值。

**Q: 一个存档能获得多少个五彩碎片？**
A: 理论上无限，但获取概率极低。普通玩家一个存档可能获得5-20个。

**Q: 五彩碎片能放进种子机吗？**
A: 不能，五彩碎片不是作物种子。

**Q: 博物馆需要五彩碎片吗？**
A: 可以捐赠但不是必须的，不影响博物馆完成度。
