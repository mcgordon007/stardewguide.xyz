# 恐龙蛋攻略

恐龙蛋（Dinosaur Egg）是星露谷物语中最稀有的古物之一。它可以孵化成恐龙、捐赠博物馆、制作恐龙蛋黄酱，是游戏中非常珍贵的物品。

## 获取方式

恐龙蛋的获取概率极低，以下是所有获取途径：

### 主要获取方式

| 来源 | 概率 | 说明 |
|------|------|------|
| 远古斑点（山区/采石场） | 0.6% | 用锄头挖掘 |
| 钓鱼宝箱 | 极低 | 钓鱼时出现的宝箱 |
| 怪物掉落（ Pepper Rex） | 10% | 骷髅洞穴中的恐龙怪物 |


### 其他获取方式

| 来源 | 说明 |
|------|------|
| 骷髅洞穴宝藏房 | 低概率出现 |
| 淘盘 | 极低概率 |
| 秘密森林 | 极低概率 |

> 最可靠的获取方式是在山区和采石场挖掘远古斑点，以及在骷髅洞穴击杀Pepper Rex。

## 主要用途

### 1. 孵化恐龙（最推荐）

将恐龙蛋放入**大鸡舍**的孵化器中：
- 孵化时间：约11天
- 孵化后获得一只**恐龙**
- 恐龙成年后每7天产出一个**恐龙蛋**

> 恐龙蛋售价350g，恐龙蛋黄酱售价800g，是稳定的收入来源。

### 2. 捐赠博物馆

- 可以捐赠给博物馆（Gunther）
- 捐赠后获得奖励
- 但不是完成博物馆的必要物品

### 3. 制作恐龙蛋黄酱

- 将恐龙蛋放入蛋黄酱机
- 产出**恐龙蛋黄酱**，售价800g
- 比普通蛋黄酱（190g）价值高很多

### 4. 礼物

- 恐龙蛋是部分村民的"喜欢"礼物
- 但不是"最爱"礼物

### 5. 裁缝

- 可以在缝纫机中制作恐龙服装
- 有趣的装饰用途

## 恐龙养殖

### 恐龙特点

| 属性 | 数据 |
|------|------|
| 产出 | 恐龙蛋（每7天） |
| 售价 | 350g（蛋）/ 800g（蛋黄酱） |
| 饲料 | 干草 |
| 生活环境 | 大鸡舍 |

### 养殖收益

| 项目 | 收益 |
|------|------|
| 单只恐龙月产出 | 约4个蛋 |
| 单只恐龙月收益（蛋） | 1,400g |
| 单只恐龙月收益（蛋黄酱） | 3,200g |

> 恐龙不需要特殊照顾，与普通鸡一样喂养干草即可。

## 最佳使用策略

### 第一个恐龙蛋

**强烈推荐用于孵化恐龙**，原因：
1. 恐龙可以持续产出恐龙蛋
2. 稳定的长期收入来源
3. 恐龙蛋可以制作高价值蛋黄酱

### 第二个及以后的恐龙蛋

1. **制作蛋黄酱**：恐龙蛋黄酱售价800g
2. **捐赠博物馆**：完成博物馆收藏
3. **礼物**：送给喜欢的村民

### 不建议的用途
- 直接出售（350g，价值远低于持续产出）
- 作为普通礼物（不是"最爱"礼物）

## 提高获取概率

### 最佳地点
1. **山区远古斑点**：每天检查，概率0.6%
2. **采石场远古斑点**：同样0.6%概率
3. **骷髅洞穴**：击杀Pepper Rex（10%掉落）

### 幸运加成
- 选择高幸运日挖掘远古斑点
- 吃幸运食物（幸运午餐、香辣鳗鱼）
- 佩戴幸运戒指
- 携带幸运兔脚

### 效率提升
- 每天检查所有远古斑点
- 在山区和采石场放置传送图腾
- 冬季是挖掘远古斑点的好时机（不需要浇水）

## 常见问题

**Q: 恐龙蛋能卖多少钱？**
A: 基础售价350g，但强烈建议孵化而不是出售。

**Q: 恐龙需要特殊照顾吗？**
A: 不需要，与普通鸡一样，每天喂养干草即可。

**Q: 一个鸡舍可以养几只恐龙？**
A: 大鸡舍最多12只动物，可以混合养鸡和恐龙。

**Q: 恐龙蛋可以放进种子机吗？**
A: 不可以，恐龙蛋不是作物种子。

**Q: 博物馆需要恐龙蛋吗？**
A: 可以捐赠但不是必须的，不影响博物馆完成度。

**Q: 恐龙蛋黄酱比普通蛋黄酱贵多少？**
A: 恐龙蛋黄酱800g，普通大鸡蛋蛋黄酱380g，贵420g。
