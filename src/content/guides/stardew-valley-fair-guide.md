# 星露谷展览会攻略

星露谷展览会（Stardew Valley Fair）是秋季第16天举行的重要节日。这是游戏中唯一可以购买**星之果实**（永久增加体力上限）的节日。

## 基本信息

- **日期**：秋季第16天
- **时间**：上午9:00 - 下午3:00
- **地点**：鹈鹕镇广场
- **特点**：农场当天自动浇水，可以安心参加节日

## 节日活动

### 1. Grange Display 评比

将9件物品展示在展台上，由州长和刘易斯镇长评分。评分标准：

| 评分项 | 分数 |
|--------|------|
| 物品多样性（8个类别） | 每个类别+1分 |
| 物品总价值 | 高价值物品加分 |
| 物品品质 | 铱星/金星加分 |
| 镇长刘易斯的偏好 | 特定物品额外加分 |

**8个类别：** 动物产品、工匠物品、烹饪、鱼、采集、水果、矿物、蔬菜

**高分技巧：**
- 展示8个不同类别的物品
- 优先选择铱星品质物品
- 高价值物品：钻石、铱锭、上古水果酒、五彩碎片
- 刘易斯偏好的物品：刘易斯的短裤（彩蛋，放入可获750星币）

**评分结果：**
| 排名 | 奖励 |
|------|------|
| 第一名 | 1,000星币 |
| 第二名 | 500星币 |
| 其他 | 参与奖 |

### 2. 转盘游戏（Spinning Wheel）

展览会最受欢迎的赚钱方式：

| 投注 | 赔率 | 概率 |
|------|------|------|
| 绿色 | 2:1 | 约75% |
| 橙色 | 2:1 | 约25% |

**技巧：**
- 绿色出现概率约73.3%，橙色约26.6%
- 建议每次押注绿色，稳步积累星币
- 可以将所有星币押在绿色上，快速积累

### 3. 小游戏

| 游戏 | 玩法 | 星币奖励 |
|------|------|----------|
| 钓鱼小游戏 | 在规定时间内钓鱼 | 根据钓鱼数量 |
| 弹弓小游戏 | 射击移动目标 | 根据命中数量 |
| 迷宫 | 找到出口 | 完成奖励 |

## 星币商店

用星币兑换各种奖励：

| 商品 | 星币价格 | 说明 |
|------|----------|------|
| **星之果实** | **2,000** | **永久+34体力上限（必买）** |
| 珍奇乌鸦 #1 | 800 | 稀有稻草人 |
| 稻草人 | 500 | 普通稻草人 |
| 星币帽 | 1,000 | 帽子 |
| 蓝色小鸡 | 1,000 | 解锁蓝色小鸡购买 |

> **星之果实是必买物品**，每年只能购买一次，错过等一年。

## 赚钱策略

### 最佳赚钱流程

1. **准备阶段**：
   - 带100g进入节日（最低入场费）
   - 准备好Grange Display的9件物品

2. **参加Grange Display**：
   - 展示9件高价值物品
   - 争取第一名（1,000星币）

3. **转盘赚钱**：
   - 将所有星币押在绿色上
   - 绿色概率约75%，稳步翻倍
   - 重复几次即可积累大量星币

4. **购买星之果实**：
   - 攒够2,000星币
   - 购买星之果实（永久+34体力）

### 注意事项
- 节日结束后未使用的星币**不会**保留到明年（每年清零）

- 珍奇乌鸦每年都可以购买
- 节日当天农场自动浇水，不用担心作物

## 常见问题

**Q: 星之果实每年都能买吗？**
A: 是的，每年秋季展览会都可以购买一次星之果实。

**Q: 转盘绿色真的概率更高吗？**
A: 是的，绿色概率约75%，橙色约25%。

**Q: Grange Display可以放刘易斯的短裤吗？**
A: 可以！这是一个彩蛋，放入后会获得额外750星币，刘易斯会很尴尬。

**Q: 展览会结束后星币会消失吗？**
A: 不会，未使用的星币会保留到明年。
