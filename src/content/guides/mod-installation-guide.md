# Mod Installation Guide

Stardew Valley has a vibrant modding community that has created thousands of mods to enhance and expand the game. This guide covers everything you need to know about installing and using mods.

## TL;DR

To use mods in Stardew Valley, you need SMAPI (Stardew Modding API). Download SMAPI from smapi.io, install it to your game folder, then place mod files in the Mods folder. Popular mods include Stardew Valley Expanded (SVE), UI Info Suite 2, and Automate.

## What is SMAPI?

SMAPI (Stardew Modding API) is the mod loader for Stardew Valley. It is required for most mods to function.

- **Official Website:** https://smapi.io/
- **Mod Loading:** Loads and manages all installed mods
- **Compatibility:** Checks mod versions and game compatibility
- **Error Handling:** Displays errors and helps troubleshoot

## How to Install SMAPI (Windows)

1. Download SMAPI from https://smapi.io/
2. Extract the ZIP file to a temporary folder
3. Run install.exe and select your Stardew Valley game folder
4. Launch the game using StardewModdingAPI.exe

## How to Install Mods

1. Download mods from Nexus Mods (https://www.nexusmods.com/stardewvalley)
2. Extract mod files (usually ZIP archives)
3. Place the mod folder in your game Mods folder
4. Launch SMAPI to load the mods

## Best Mods to Try

### Stardew Valley Expanded (SVE)

The largest content mod for Stardew Valley:
- 28+ new NPCs with unique stories
- 58 new locations to explore
- 278+ new events and cutscenes
- Created by FlashShifter

### UI Info Suite 2

Displays detailed information about crops, animals, and machines on the game interface.

### Automate

Automatically processes items through connected machines. Connect chests to machines for automatic processing.

### Tractor Mod

Adds a purchasable tractor (50,000g at Robin shop) for automated farming tasks.

## Troubleshooting

- **Game won't start:** Check SMAPI console for errors
- **Mod not loading:** Verify mod is in correct folder
- **Game crashes:** Remove problematic mod
