# 战斗技能攻略

战斗技能影响武器伤害、生命值和战斗效率。

## 如何提升

- **击杀怪物**：+2-5战斗经验
- **矿井战斗**：主要经验来源
- **骷髅洞穴战斗**：后期主要经验来源

## 等级奖励

| 等级 | 奖励 |
|------|------|
| 1 | 解锁石剑配方 |
| 2 | 解锁生命药水配方 |
| 3 | 解锁英雄药水配方 |
| 4 | 解锁钢制小剑配方 |
| 5 | 职业选择 |
| 6 | 解锁铱剑配方 |
| 8 | 解锁生命花药水配方 |
| 9 | 解锁银河剑购买权限 |
| 10 | 职业选择 |

## 5级职业选择

### 战士（Fighter）- 攻击+10%
### 游侠（Scout）- 暴击率+50%

> **推荐战士**，稳定的10%攻击加成比不可靠的暴击更实用。

## 10级职业选择

### 武士（Brute）- 攻击+15%（需战士）
### 防御者（Defender）- 生命值+25（需战士）

> **推荐武士**，+25%攻击（与战士叠加）非常强力。

## 常见问题

**Q: 战士和游侠哪个好？** A: 战士更实用。
**Q: 武士和防御者哪个好？** A: 武士更好。
