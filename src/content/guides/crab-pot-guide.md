# 蟹笼攻略

蟹笼（Crab Pot）是星露谷物语中的钓鱼辅助设备，可以自动捕获水中的各种生物。它是完成钓鱼收集包和获取特定材料的重要工具。

## 如何解锁与制作

### 基础蟹笼

- **解锁条件**：钓鱼技能达到3级
- **制作材料**：40个木头 + 3个铁锭


## 使用方法

1. 制作蟹笼
2. 将蟹笼放入水中（河流、湖泊、海洋均可）
3. 放入鱼饵（可选，有鱼饵时不会捕获垃圾）
4. 第二天来收集捕获物
5. 重新放入水中继续工作

> 蟹笼需要每天手动收集和重新放置。

## 钓鱼职业选择

钓鱼技能5级和10级的职业选择影响蟹笼收益：

### 5级选择

| 职业 | 效果 |
|------|------|
| **渔夫（Fisher）** | 鱼价值+25% |
| **捕蟹者（Trapper）** | 蟹笼制作材料减半（25木头+2铜锭）|

> **推荐选择捕蟹者**，省去每天制作鱼饵的麻烦。

### 10级选择（需选择捕蟹者）

| 职业 | 效果 |
|------|------|
| **水手（Mariner）** | 蟹笼不再捕获垃圾 |
| **诱饵大师（Luremaster）** | 蟹笼不再需要鱼饵（自动补饵）|

> **推荐选择水手**，确保蟹笼只捕获有价值的物品。

## 实用技巧

### 自动化建议
- 选择捕蟹者职业后不需要鱼饵
- 选择水手职业后不捕获垃圾
- 将蟹笼放在农场附近，方便每天收集

### 收集包用途
蟹笼捕获物是多个社区中心收集包的必需品：
- **鱼缸收集包**：螃蟹、龙虾、小龙虾、蜗牛等
- **厨师收集包**：各种海鲜

### 烹饪用途
- 蟹笼捕获物可用于多种烹饪配方
- 龙虾浓汤、海鲜杂烩汤等

### 与工匠职业
- 如果选择工匠职业（农耕10级），蟹笼捕获物售价+40%
- 但通常钓鱼职业更适合蟹笼玩家

## 常见问题

**Q: 蟹笼可以放在农场外的水里吗？**
A: 可以，任何水域都可以放置。

**Q: 蟹笼会被冲走吗？**
A: 不会，蟹笼固定在水中位置。

**Q: 冬天蟹笼还能工作吗？**
A: 可以，蟹笼全年工作。

**Q: 蟹笼捕获物会腐烂吗？**
A: 不会，收集后永久保存。

**Q: 捕蟹者职业真的不需要鱼饵吗？**
A: 是的，选择捕蟹者后蟹笼不再需要鱼饵也能正常工作。
