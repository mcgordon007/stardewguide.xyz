# 古代种子攻略

古代种子（Ancient Seed）是星露谷物语中最珍贵的种子之一，种植后产出**上古水果**（Ancient Fruit）。上古水果是游戏中最赚钱的作物，配合酒桶和地窖陈酿可以产生巨额利润。

## 获取方式

古代种子的获取方式特殊，以下是所有途径：

### 主要获取方式

| 来源 | 概率 | 说明 |
|------|------|------|
| 远古斑点（任意地点） | 0.7% | 用锄头挖掘 |
| 钓鱼宝箱 | 极低 | 钓鱼时出现的宝箱 |
| 怪物掉落（虫子类） | 极低 | 洞穴蝇、蛆等 |
| 种子机 | — | 将上古水果放入种子机 |

### 博物馆奖励

- 将古代种子捐赠给博物馆（Gunther）
- 捐赠后获得 **上古种子配方** + 1包上古种子
- 这是获得上古种子的最可靠方式

### 种子机产出

- 将上古水果放入种子机
- 产出1-3个上古种子（平均1.97个）
- 是后期大量获取种子的主要方式

## 上古水果特点

| 属性 | 数据 |
|------|------|
| 生长周期 | 28天（首次种植） |
| 重复收获周期 | 7天 |
| 基础售价 | 550g |
| 季节 | 春季、夏季、秋季（或温室/姜岛全年） |
| 是否持续收获 | 是 |

> 上古水果种植后每7天收获一次，直到季节结束或作物被铲除。

## 种植策略

### 温室种植（最推荐）

温室是种植上古水果的最佳地点：

- **不受季节限制**：全年生长和收获
- **不会枯萎**：作物永久存在
- **配合酒桶**：温室内或附近放置酒桶

#### 温室布局

- 温室内可种植约120块地
- 使用铱级洒水器自动浇水
- 旁边放置酒桶，收获后立即酿造

### 姜岛农场种植

姜岛农场是另一个理想地点：

- **不受季节限制**
- **没有乌鸦**
- **大面积耕地**：约941块可耕地

### 户外种植

如果在户外种植（春季/夏季/秋季）：

- 春季第1天种植，到秋季结束可收获多次
- 季节更替时作物会枯萎
- 需要每年重新种植

## 最佳配置

### 推荐设置

| 项目 | 建议 |
|------|------|
| 种植地点 | 温室 + 姜岛农场 |
| 洒水器 | 铱级洒水器 |
| 肥料 | 顶级肥料（提升品质） |
| 酒桶数量 | 越多越好（建议200+） |
| 地窖 | 189个小桶全部用于陈酿 |

### 时间规划

- **第1天**：种植上古种子
- **第28天**：首次收获
- **第28天**：将上古水果放入酒桶酿造
- **第35天**：酒酿造完成，收入地窖陈酿
- **第91天**：铱星酒陈酿完成，收获出售
- **每7天**：持续收获上古水果

## 种子扩繁策略

### 初期（1-10个种子）

1. 获得第一个古代种子后，捐赠博物馆获得配方和种子
2. 在温室种植
3. 收获后部分上古水果直接出售，部分放入种子机
4. 逐步扩大种植规模

### 中期（10-50个种子）

1. 温室全部种满上古水果
2. 大量制作酒桶
3. 收获的上古水果一半酿酒，一半放入种子机

### 后期（50+个种子）

1. 姜岛农场大面积种植
2. 酒桶数量最大化
3. 地窖189个小桶全部陈酿上古水果酒
4. 实现完全自动化的高额收入

## 实用技巧

### 早期获取古代种子
- 每天检查所有远古斑点
- 冬季是挖掘远古斑点的好时机
- 钓鱼时留意宝箱

### 温室优先
- 优先在温室种植上古水果
- 温室作物不会枯萎，永久产出
- 配合自动洒水器，几乎零维护

### 酒桶规划
- 上古水果每7天收获一次
- 酒酿造需要7天
- 地窖陈酿需要56天
- 建议酒桶数量 = 种植面积 x 2（保证酿造和陈酿不积压）

### 工匠职业
- 农耕10级务必选择工匠职业
- +40%售价让上古水果酒利润大幅提升
- 铱星上古水果酒售价6,468g

## 常见问题

**Q: 古代种子在哪里获得？**
A: 挖掘远古斑点（0.7%概率）、钓鱼宝箱、捐赠博物馆后获得配方。

**Q: 上古水果可以种在冬季吗？**
A: 户外不行，但温室和姜岛农场全年可种植。

**Q: 上古水果酒是最赚钱的吗？**
A: 是的，铱星上古水果酒配合工匠职业售价6,468g，是游戏中最赚钱的产品。

**Q: 一个上古种子可以收获几次？**
A: 在温室或姜岛可以无限收获（每7天一次），户外种植到季节结束。

**Q: 上古水果可以放入种子机吗？**
A: 可以，平均产出1.97个种子，是扩繁的主要方式。

**Q: 温室可以种多少上古水果？**
A: 约120块地，使用铱级洒水器可以全覆盖。
