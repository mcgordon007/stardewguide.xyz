# 农场地图选择与布局攻略

选择农场地图是星露谷物语中最重要的开局决策之一。游戏提供8种不同的农场地图，各有特色。

## 8种农场地图对比

| 农场 | 可耕地 | 特色 | 适合玩家 |
|------|--------|------|----------|
| 标准农场 | 最大 | 无特殊限制，最平衡 | 新手首选 |
| 河边农场 | 中等 | 大量水域，适合钓鱼 | 钓鱼爱好者 |
| 森林农场 | 中等 | 大量树木，采集物丰富 | 采集/觅食玩家 |
| 山顶农场 | 中等 | 矿石丰富，采矿方便 | 采矿爱好者 |
| 荒野农场 | 中等 | 夜间怪物出现 | 战斗爱好者 |
| 四角农场 | 大 | 4个分区，适合多人 | 多人游戏 |
| 海滩农场 | 小 | 海边，特殊采集物 | 休闲玩家 |
| 草原农场 | 大 | 蓝色草，动物友好 | 养殖爱好者 |

## 标准农场（Standard Farm）

### 特点
- **最大可耕地面积**：约3,427格
- 无特殊限制
- 最平衡的选择
- 适合任何玩法

### 布局建议
- **北部**：作物种植区（配合洒水器）
- **中部**：酒桶/小桶加工区
- **南部**：动物谷仓/鸡舍
- **边缘**：树木种植区（树液采集器）

> **新手强烈推荐选择标准农场。**

## 河边农场（Riverland Farm）

### 特点
- 大量水域（适合钓鱼）
- 可耕地被河流分割
- 适合蟹笼放置

### 布局建议
- 利用小岛种植作物
- 水域放置大量蟹笼
- 钓鱼为主要收入来源

## 森林农场（Forest Farm）

### 特点
- 大量树木和采集物
- 西部有采集区（产出季节性采集物）
- 适合觅食技能提升

### 布局建议
- 保留西部采集区
- 种植树木获取树液
- 适合树液采集器流

## 山顶农场（Hill-top Farm）

### 特点
- 西南部有采矿区（产出矿石）
- 矿石节点定期刷新
- 适合采矿技能提升

### 布局建议
- 利用采矿区获取矿石
- 减少去矿井的频率
- 适合矿工职业玩家

## 荒野农场（Wilderness Farm）

### 特点
- 夜间怪物出现（根据战斗等级）
- 怪物掉落物品
- 适合战斗技能提升

### 布局建议
- 注意夜间安全
- 利用怪物掉落获取资源
- 适合战士职业玩家

## 四角农场（Four Corners Farm）

### 特点
- 4个独立分区
- 适合多人游戏（每人一个分区）
- 可耕地面积大

### 布局建议
- 每个分区专精一种玩法
- 分区1：作物
- 分区2：动物
- 分区3：酿酒
- 分区4：树木/装饰

## 海滩农场（Beach Farm）

### 特点
- 海边位置
- 特殊采集物（海胆、珊瑚等）
- 洒水器在沙地上效率降低

### 布局建议
- 利用海边采集物
- 在可耕地使用洒水器
- 适合休闲玩法

## 草原农场（Meadowlands Farm）

### 特点
- 1.6版本新增
- 蓝色草（动物更喜欢）
- 动物好感度增长更快

### 布局建议
- 大量养殖动物
- 蓝色草提升动物产出
- 适合牧场主职业

## 通用布局技巧

### 洒水器布局

**铱级洒水器**（覆盖24格）：
```
  C C C C C
  C S S S C
  C S W S C
  C S S S C
  C C C C C
```
W = 铱级洒水器，S = 可耕地，C = 通道

### 温室布局
- 使用铱级洒水器覆盖最大面积
- 旁边放置酒桶，收获后立即酿造
- 种植上古水果或杨桃

### 酒桶布局
- 在农舍地窖放置189个小桶
- 在温室旁边放置额外酒桶
- 姜岛农场也可以放置酒桶

### 动物区布局
- 谷仓和鸡舍靠近农舍
- 门口留出空地让猪觅食
- 放置自动采集器减少工作量

## 常见问题

**Q: 新手应该选择哪个农场？**
A: 标准农场，最大可耕地，无特殊限制。

**Q: 农场选择后可以更改吗？**
A: 不可以，选择后无法更改。

**Q: 哪个农场最赚钱？**
A: 标准农场和四角农场，可耕地面积最大。

**Q: 草原农场的蓝色草有什么特殊？**
A: 动物更喜欢蓝色草，好感度增长更快，产出品质更高。
